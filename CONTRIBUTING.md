
# Contributing examples

## There are a few requirements you must followave to fulfill

1. thumbnails must be 16:9 (preferrably 640x360 for consistency, but not a requirement)
2. thumbnails must be .jpg
3. add yourself as the author on the .ron files
4. add the new examples to the examples/README.md
5. project names are recommended to be Title Case (something like "Acid Pool" and "Windy Snow")
